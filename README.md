# OpenLayers-PlatForm
OpenLayers를 기반으로 Web GIS 서비스 개발 시 자주 사용되는 기능들을 모듈화한 스크립트 라이브러리. <br>
A script library that modularized frequently used functions when developing Web GIS services based on OpenLayers
